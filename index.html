Welcome to Cybersecurity Awareness Hub

Learn about common hacking techniques and how to defend against them.

Phishing

Phishing is a fraudulent attempt to obtain sensitive information by disguising as a trustworthy entity.

Defense: Always verify email senders and avoid clicking suspicious links.

SQL Injection

Attackers exploit vulnerabilities in database queries to access or manipulate data.

Defense: Use parameterized queries and input validation.
