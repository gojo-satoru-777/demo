Welcome

Cybersecurity is everyone’s responsibility. Learn about threats and how to stay safe online.

Phishing

Phishing attacks trick users into revealing sensitive information by pretending to be trusted sources.

  • ⚠️ Fake emails or websites mimicking banks/social media.
  • ✅ Always check URLs before clicking.
  • ✅ Enable multi-factor authentication.

SQL Injection

SQL Injection exploits insecure database queries to steal or manipulate data.

  • ⚠️ Attackers insert malicious SQL into input fields.
  • ✅ Use parameterized queries.
  • ✅ Validate and sanitize user inputs.

Malware

Malware is malicious software designed to damage or gain unauthorized access to systems.

  • ⚠️ Viruses, worms, ransomware.
  • ✅ Keep software updated.
  • ✅ Use antivirus and firewalls.

Password Attacks

Hackers attempt to crack or steal passwords to gain access.

  • ⚠️ Brute force, dictionary attacks, credential stuffing.
  • ✅ Use strong, unique passwords.
  • ✅ Enable password managers.

DDoS (Distributed Denial of Service)

DDoS attacks flood servers with traffic, making them unavailable.

  • ⚠️ Botnets overwhelm websites.
  • ✅ Use load balancers and firewalls.
  • ✅ Monitor traffic patterns.

Man-in-the-Middle (MITM)

MITM attacks intercept communication between two parties.

  • ⚠️ Attackers eavesdrop on unsecured Wi-Fi.
  • ✅ Use HTTPS and VPNs.
  • ✅ Avoid public Wi-Fi for sensitive tasks.
